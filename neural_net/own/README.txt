The code in this folder is mainly based on the code written if you follow along with the Make Your Own Neural Network book by Tariq Rashid. You can buy a copy on Amazon (https://www.amazon.com/Make-Your-Own-Neural-Network-ebook/dp/B01EER4Z4G) or check out the code on GitHub (https://github.com/makeyourownneuralnetwork/makeyourownneuralnetwork/). I highly suggest checking them out.
